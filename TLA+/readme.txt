File format explanation:

<Protocol Name>_<Number of Participants>.tla

The file ExtendedEuclid.tla and ModExp.tla contains the code for inverse exponent function and modular exponentiation function respectively. Make sure p is a prime number and g is the generator of Zp.

The hyperbook of TLA+ language by Leslie Lamport is available at

https://lamport.azurewebsites.net/tla/hyperbook.html

and some materials regarding TLA+ is also available at

http://lamport.azurewebsites.net/tla/papers.html
